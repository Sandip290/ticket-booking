# Ticket Booking System - Full-Stack Java Application

Ticket Booking System is a full-stack application built to simulate a real-world ticket reservation workflow with user management and booking operations.

## Tech Stack
- Java
- Spring Boot
- MySQL
- REST APIs
- HTML / CSS (UI)

## Key Features
- User registration and authentication
- Ticket booking and cancellation workflows
- Backend APIs for booking operations
- Database integration using MySQL

## What I Worked On
- Designed backend logic for ticket booking workflows
- Implemented REST APIs using Spring Boot
- Integrated MySQL for persistent data storage
- Focused on clean service-layer design and validation
